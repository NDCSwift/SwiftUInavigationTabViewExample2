# 🗂️ SwiftUI Navigation & TabView
A practical SwiftUI example showing how to combine NavigationStack and TabView for multi-screen app navigation.

---

## 🤔 What this is
This project demonstrates how to structure a SwiftUI app with both tab-based and stack-based navigation. It shows how TabView and NavigationStack work together, how to pass data between screens, and how to manage navigation state cleanly. A solid starting point for any multi-screen iOS app.

## ✅ Why you'd use it
- **TabView + NavigationStack** — Shows the idiomatic iOS 16+ navigation pattern with tabs and push navigation
- **State management** — Demonstrates how to manage navigation paths and share data across tabs
- **Real-world structure** — Organized the way a production SwiftUI app would be

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/SwiftUInavigationTabViewExample2.git
cd SwiftUInavigationTabViewExample2
```

### 2. Open in Xcode
Double-click `navigationTabViewExample2.xcodeproj`.

### 3. Set Your Development Team
In Xcode: **TARGET → Signing & Capabilities → Team** — select your team.

### 4. Update the Bundle Identifier
Change `com.example.MyApp` to a unique reverse-domain ID.

## 🛠️ Notes
- Uses NavigationStack (iOS 16+), not the deprecated NavigationView.
- If you see a code signing error, verify Team and Bundle ID are set.

## 📦 Requirements
- Xcode 15+
- iOS 16+
